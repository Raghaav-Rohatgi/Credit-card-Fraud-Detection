# Credit-card-Fraud-Detection
•	Developed a robust pipeline for fraud detection, processing over 10,000+ records through data cleaning, outlier handling, and balancing the dataset. Achieved 92% accuracy, F1 score of 0.85, and a specificity of 0.91 as part of the classification report.
•	Compared classifiers, including Logistic Regression, Random Forest, Decision Tree, and K-Nearest Neighbors, with Random Forest achieving 93% accuracy.
•	Optimized model performance by encoding categorical variables, standardizing features, and reducing dimensionality by 15%, leading to improved precision (0.88), recall (0.87), and overall model accuracy.
